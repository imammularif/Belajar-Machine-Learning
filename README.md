# Belajar-Machine-Learning

Pada repository ini, saya mempraktikkan bagaimana sebuah dataset diolah menggunakan teknik Machine Learning.

Proses yang dilakukan meliputi:
- Loading data mentah
- Data cleaning (menghapus duplikasi)
- Identifikasi dan penanganan missing values
- Deteksi dan penanganan outliers
- Normalisasi dan standarisasi data

Setelah melalui tahapan tersebut, dataset menjadi lebih bersih dan siap digunakan untuk proses modeling Machine Learning.
